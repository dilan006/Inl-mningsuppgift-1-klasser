Om programmet

Det här C++-programmet läser in mätvärden från två sensorer (temperatur och luftfuktighet).
Det kan spara mätdata, visa statistik och läsa/spara data till fil.



Vad programmet kan göra
	•	Skapa två sensorer
	•	Ta nya mätvärden
	•	Visa alla mätvärden
	•	Räkna ut statistik (medel, min, max, antal)
	•	Spara mätvärden till fil
	•	Läsa mätvärden från fil
	•	Använda en meny i terminalen



Kort hur koden fungerar
	•	Sensor → skapar ett slumpmässigt mätvärde
	•	Measurement → lagrar ett mätvärde
	•	MeasurementStorage → sparar alla värden och räknar statistik
	•	main() → visar menyn och kör programmet
